# 美术素材处理工具 V1 可执行开发计划

- 计划版本：v1.1
- 日期：2026-06-28
- 里程碑：V1
- 平台：Linux（当前实现与验收）

## 1. V1 目标

在 Linux 平台交付可稳定运行的素材处理应用，覆盖：

1. 图片切分（3 模式 + 预览）
2. 图片缩放（仅缩小）
3. 抠图（三档算法 + ONNX Runtime）
4. 帧动画（播放/FPS/删帧/拖拽）
5. 批量导出（统一格式 + 前缀序号命名）
6. 插件机制（运行时动态加载）
