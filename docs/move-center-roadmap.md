# 移动画面（居中）路线图

## 已实现

- V1：按前景包围盒中心移动到画布中心（单张/批量）。
- V1.5：批量支持“中位锚点”策略，统一多帧内容位置以降低动画抖动。
- 位置：`调整图片` tab 的 `移动` 模块。
- 当前交互：
   - `应用`：处理当前素材
   - `批量`：处理当前素材池全部素材
   - `前景阈值(alpha)`：用于前景区域检测
   - 批量完成后会显示统计摘要：平均偏移、最大偏移、有效前景检测数
- 一致性说明：当策略为“中位锚点”时，`应用` 与 `批量` 使用同一批次锚点基准。

## 待讨论（V2，计划放入动画 tab）

1. 参考帧策略
   - 参考帧来源：首帧 / 当前帧 / 指定帧
   - 对齐目标：仅平移或允许轻微缩放（默认仅平移）

2. 锚点定义
   - 包围盒中心、重心、底边中心（角色动画常用）
   - 是否支持横纵轴分离锚点（例如 x 用重心，y 用底边）

3. 抖动抑制
   - 全局平滑：移动量低通滤波或滑动窗口
   - 异常帧处理：阈值裁剪、回退到邻帧估计

4. 失败回退
   - 前景检测失败时是否回退到上一帧偏移
   - 透明度/噪声场景阈值自适应策略

5. 交互与可视化
   - 时间线上显示每帧 dx/dy 曲线
   - 一键“预览稳定前后对比”
   - 批量执行前后的统计摘要（均值/方差/最大偏移）

6. 与导出链路关系
   - 动画 tab 应用稳定化后，是否写回 `调整图片` 结果池
   - 导出时优先级（动画稳定结果 vs 调整图片结果）
