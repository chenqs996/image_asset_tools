# 美术素材处理工具 V1 验收测试用例文档

- 文档版本：v1.0
- 日期：2026-06-28
- 适用版本：V1（Linux）
- 对应计划：`docs/V1可执行开发计划.md`

## 1. 验收目标

验证 V1 是否满足以下交付能力：

1. 素材导入（PNG/JPG/WebP/BMP）
2. 切分（三模式 + 预览 + 坐标结果一致）
3. 缩放（仅缩小）
4. 抠图（三档算法 + 单图/批量 + 单张例外参数）
5. 帧动画（播放/FPS/删帧/拖拽）
6. 批量导出（命名规则 + 统一格式）
7. 插件动态加载与基础稳定性

## 2. 测试环境

- OS：Linux
- Node.js：>= 20
- npm：>= 10
- 命令执行目录：`app/`

## 3. 测试数据准备

建议准备以下样本：

- `sample_sprite_sheet.png`（规则网格图）
- `sample_irregular_sheet.png`（不规则切分线）
- `sample_subject_white_bg.jpg`（纯色背景）
- `sample_subject_checker_bg.png`（灰白方格背景）
- `frame_001.png ... frame_010.png`（帧动画序列）

## 4. 自动化验收项（脚本）

通过脚本自动执行：

1. 代码规范检查（lint）
2. 单元测试（vitest）
3. 生产构建（build）
4. 核心文件存在性检查（插件清单、关键页面）
5. 插件 manifest JSON 可解析性检查
6. 构建产物存在性检查（`dist/index.html`）

## 5. 手工验收用例

### A. 导入与素材管理

#### ATC-001 导入格式支持
- 前置条件：应用已启动
- 步骤：导入 PNG/JPG/WebP/BMP 四类文件
- 预期：素材列表均正确显示，含尺寸与格式信息

#### ATC-002 素材选中联动
- 步骤：在导入页切换选中素材，再进入处理页
- 预期：处理页“选择素材”与预览对象一致

### B. 切分功能

#### ATC-101 指定尺寸切分
- 步骤：模式切到“指定单片尺寸”，输入 `256x256`
- 预期：网格预览更新，切片数量与预览一致

#### ATC-102 指定数量切分
- 步骤：模式切到“指定横竖数量”，输入 `4x3`
- 预期：单片尺寸自动反算正确，网格显示正确

#### ATC-103 自动切分线提取
- 步骤：模式切到“自动/手动切分线”，点击自动识别
- 预期：识别出 X/Y 线条并可见

#### ATC-104 手动线条增删与拖拽
- 步骤：在预览图添加线条、删除线条、拖拽线条
- 预期：线条位置与切片坐标清单实时更新

### C. 缩放功能（仅缩小）

#### ATC-201 按比例缩放预览
- 步骤：输入比例 `1,0.5,0.25`
- 预期：生成三档预览尺寸，`1x`为原始尺寸

#### ATC-202 放大拦截
- 步骤：输入比例 `2`
- 预期：该项标记为“禁止放大”

### D. 抠图功能

#### ATC-301 抠图算法切换
- 步骤：依次切换 AI通用/纯色色键/灰白方格
- 预期：可正常切换并执行单图预览

#### ATC-302 单图抠图预览
- 步骤：调整 threshold/smooth/denoise/feather，执行单图预览
- 预期：右侧结果图更新，视觉可辨识

#### ATC-303 批量抠图
- 步骤：点击“批量抠图”
- 预期：状态信息显示进度，全部样本完成

#### ATC-304 单张例外参数
- 步骤：对当前图点击“设为例外参数并重跑”
- 预期：例外参数清单出现该图记录，结果更新

### E. 帧动画功能

#### ATC-401 播放与暂停
- 步骤：点击播放，再点击暂停
- 预期：帧预览按 FPS 切换并可停止

#### ATC-402 FPS 调整
- 步骤：设置 FPS 为 6 与 24
- 预期：播放速度变化明显

#### ATC-403 删帧与拖拽排序
- 步骤：删除中间帧；拖拽最后一帧到开头
- 预期：总帧数变化正确，顺序即时生效

### F. 批量导出

#### ATC-501 命名规则生效
- 步骤：设置 `prefix=hero`, `start=1`, `digits=3`, `suffix=bg`
- 预期：导出名形如 `hero_001_bg.xxx`

#### ATC-502 统一输出格式
- 步骤：分别选择 PNG/BMP/WebP 导出
- 预期：同一批次导出文件扩展名一致

#### ATC-503 选择导出子集
- 步骤：仅选中 3 张素材导出
- 预期：仅下载 3 个目标文件

### G. 插件机制

#### ATC-601 插件清单加载
- 前置条件：`public/plugins/plugins.manifest.json` 存在
- 步骤：重启应用，查看“已注册算法”
- 预期：包含内置算法与示例插件项
